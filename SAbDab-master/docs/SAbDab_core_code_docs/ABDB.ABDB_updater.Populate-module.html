populate_part(code, method_name='', method_args=[], method_kwargs={}, dbpath='', overwrite=False)

source code 

Partial populate function This is used to when adding new information to the entries of the database. It means that it is not necessery to re-calculate all the information in the database.

Parameters:
  • code - The pdb code of an entry you want to populate details for
  • method_name - The name of the method of the populater object that you want to call
  • dbpath - The path to the database you want to populate
  • overwrite - Whether to allow the populator to overwrite what is already their
  • method_args - additional arguments to the method
  • method_kwargs - additional keyword arguments to the method.
Returns: tuple

e.g. populate_part("12e8", "_partial_test", ["chicken"],method_kwargs={"something_else":"cluck"},dbpath="", overwrite=True)

Tuple of success_flag, code, error_message
