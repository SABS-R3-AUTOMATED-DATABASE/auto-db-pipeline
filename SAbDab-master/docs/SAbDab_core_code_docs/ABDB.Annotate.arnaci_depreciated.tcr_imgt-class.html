number(self, seq, profile)

 

Use the IMGT scheme to number the sequence. This algorithm is based on heuristic to mimic what we see from IMGT numbered sequences. Limitations are the numbering of the insertions at cdr3. Currently insertions are made at 111 only - this is inconsistent with IMGT who spread the insertions over 111 112.
