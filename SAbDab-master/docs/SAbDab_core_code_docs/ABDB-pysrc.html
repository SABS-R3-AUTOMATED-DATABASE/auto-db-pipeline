 13  To install the python interface package and script: 
 14      1. Put the ABDB python package (./lib/python/ABDB) in your PYTHONPATH 
 15      2. Put the SAbDab (./bin/SAbDab) script in your PATH 
 16      3. Copy over the data from the central repository to your machine's data dir. 
 17      4. Run the setup script: SAbDab --setup 
 18       
 19  N.B. For each machine you use the API on you will have to re-run the setup process. 
 20  These settings will be saved for that machine.  
 21   
 22   
 23  See data/README for a summary of the data contents. 
 24   
 25   
 26  To use ABDB package in a python program of your own: 
 27      import ABDB 
 28   
 29  and to use the database api: 
 30      from ABDB import database 
 31   
 32  For documentation of the API open the ../docs/SAbDab_core_code_docs/index.html 
 33      or  
 34  help(database) in python

 40      o A local copy of the data files.

 43          o numpy 
 44          o scipy

 65   
 66  Example python api useage: 
 67   
 68      # Load information about the structure 12e8 from the database 
 69   
 70      # Import the database 
 71      from ABDB import database 
 72   
 73      # Fetch a pdb_details object for 12e8 
 74      p = database.fetch("12e8")  # p is a pdb_details object for 12e8 
 75   
 76      # Get the sequences for the chains in the structure 
 77      sequences = p.get_sequence() 
 78   
 79      # Use help(p) to see the other methods that can be used. 
 80   
 81      # Get a list of fab_details (fabs/paired chains) objects: 
 82      fabs = p.get_fabs() 
 83   
 84      # Get the lengths of the CDRs for the first fab in the list: 
 85      cdr_lengths = fabs[0].get_CDR_lengths() 
 86   
 87      # This is equivalent to: 
 88      cdr_lengths = p["HL"].get_CDR_lengths() 
 89   
 90      # Get the structure of 12e8: 
 91      s = p.get_structure() # returns a biopython structure object that has been put into an "antibody context". 
 92      # see help(s) for more information. 
 93   
 94      # Save only the variable region oordinates of the fab "HL" in a file named 12e8_HL_fv.pdb. (Resid's are chothia numbered) 
 95      from ABDB.AbPDB.Select import fv_only # import a selector (see help(ABDB.AbPDB.Select) for more info ) 
 96      s[0]["HL"].save(output="12e8_HL_fv.pdb", selection=fv_only() ) 
 97   
 98      # Similarly, just the VH domain of chain H 
 99      s[0]["HL"]["H"].save(output="12e8_H_VH.pdb", selection=fv_only() ) 
100   
101      Any errors/bugs/un- or badly documented functions,  please let James know
