Returns:

get_CDR_clusters(self, cdr=None, definition='chothia', cutoff='0.5')

source code 

Get the cdr cluster for a cdr or all of them.

Parameters:
  • cdr - The cdr you wish to return. Leave as None to return all Choose from H1,H2,H3,L1,L2 or L3
  • definition - The CDR characterisation you wish to use. Choose from chothia, kabat or contact
  • cutoff - The UPGMA cutoff that should be used. This ensures no two structures in a cluster have an RMSD greater than this value. Choose from 0.5, 0.75,1.0 and 1.5

    The cluster returned will be the cluster id according to the sabdab clustering for the length. This will change each time more structures are added. Use get_same_cluster_structures to get structures with similar loops.

get_same_cluster_structures(self, cdr=None, definition='chothia', cutoff='0.5')

source code 

Get the identifiers of structures with the same CDR cluster as this fab.

Parameters:
  • cdr - The cdr you wish to return. Leave as None to return all Choose from H1,H2,H3,L1,L2 or L3
  • definition - The CDR characterisation you wish to use. Choose from chothia, kabat or contact
  • cutoff - The UPGMA cutoff that should be used. This ensures no two structures in a cluster have an RMSD greater than this value. Choose from 0.5, 0.75,1.0 and 1.5

    The returned list (or dictionary of lists) will contain the structures in the database that are in the same cluster.

get_orientation_angles(self, angle='')

source code 

Get the orientation angles of the fv region. These are the angles between the VH-VL domain as described in: ABangle: Characterising the VH-VL orientation in antibodies. Dunbar J, Fuchs A, Shi J and Deane CM. PEDS. 2013

Parameters:
  • angle - Optional argument for the angle returned. Choose one of "HL","HC1","HC2","LC1","LC2" or "dc".
Returns: float or dict of floats. Empty dict if orientation angles cannot be found/calculated.
Either a dictionary of the orientation angles or the specific value of the angle request.
