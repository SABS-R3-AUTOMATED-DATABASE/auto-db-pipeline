Change Log:
  • unnumbered list now contains the id's of the unnumbered residues. Unnumbered residues are now yielded by iterating over the list.
  • The child residues now have a chothia numbered flag (True if chothia numbered false otherwise).
  • non-amino acid HETATMs now included in the children. (and reference in unnumbered)
  • ABchain.antigen is now a list. Each member of the list is a bound antigen. This handles multi-chain antigens.
