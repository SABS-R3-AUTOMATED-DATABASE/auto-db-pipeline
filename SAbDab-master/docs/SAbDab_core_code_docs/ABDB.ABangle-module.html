select_by_angle(ranges, workspace)

source code 

Select structures in using ranges of their angles. If a range is not specified for an angle, then it will not discriminate.

Parameters:
  • ranges - A dictionary with the angle as keys and list of (min, max) angle tuples for selection.
  • workspace - A dictionary containing pdb_details objects from which the selection should be made.
